# Vikipedia Codex

Это пример структуры продвинутого сайта наподобие Википедии, реализованного на Node.js с использованием Express и SQLite.

## Запуск проекта

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите сервер:
   ```bash
   npm start
   ```

Сервер будет доступен по адресу `http://localhost:3000`.
