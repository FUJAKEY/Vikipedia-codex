# Vikipedia Codex

Это пример структуры продвинутого сайта наподобие Википедии, реализованного на Node.js с использованием Express и JSON.
На страницах доступен поиск статей и выдвижное меню с категориями.
Стили вдохновлены сайтом PlayStation и адаптированы для мобильных устройств.

## Запуск проекта

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите сервер:
   ```bash
   npm start
   ```

Данные статей хранятся в файле `src/db/wiki.json`, который создаётся автоматически.
Сервер будет доступен по адресу `http://localhost:3000`.
